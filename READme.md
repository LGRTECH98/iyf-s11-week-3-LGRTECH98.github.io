## Project Description
This project is a personal portfolio website built using HTML, CSS, and JavaScript. It introduces who I am, showcases my programming skills, highlights my projects, and provides my contact information.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Git
- GitHub

## Features
- Responsive personal portfolio website
- About Me section
- Skills section
- Projects section
- Downloadable CV
- Contact form
- Social media links
- Dark mode
- Animations

## Project Structure

```
portfolio/
```

## How to Run

1. Clone the repository

```bash
git clone https://github.com/lgrtech98/portfolio.git
```

2. Open the project folder.

3. Open `index.html` in your web browser.
